# Programming 4 game engine

Programming 4 is a course in the Howest DAE major GD. This is a course we get during our second semester of the second year. One of our main assignments is making a game engine. This repo helps us take care of our project and show the steps we did starting from the [Minigin](https://github.com/avadae/minigin/releases/tag/1.3.0) from avadae.

# Goal

The goal is to make a good game engine that we can then use to make a given 80's arcade game with.
